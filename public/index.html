Live Server Project

Real-time live server for web development

About This Project

This project is a live server designed to help web developers by providing automatic reloading whenever there are changes in HTML, CSS, or JavaScript files.

Features

  • Automatic reload on file changes
  • Cross-browser compatibility
  • Customizable server settings
  • Static file serving

How to Use

  1. Clone the repository: git clone https://github.com/yourusername/live-server-project.git
  2. Install dependencies: npm install
  3. Start the server: npm start
  4. Open your browser and visit http://localhost:8080
